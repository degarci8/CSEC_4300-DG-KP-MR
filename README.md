## Malware Analyzed

## Project Overview
This project analyzes the ShadowV2 malware, a Mirai-based IoT botnet. The objective is to understand how the malware operates, how it behaves during execution, and how it can be detected or mitigated.

## Project Phases
1. Malware Analysis Environment Setup
2. Static Malware Analysis
3. Assembly-Level Code Analysis
4. Dynamic and Memory Analysis
5. Malware Behavior and Defense

## Documentation
All analysis documentation is maintained in the GitHub Wiki.

## Important Note
Malware binaries are **NOT stored in this repository** for security reasons.
